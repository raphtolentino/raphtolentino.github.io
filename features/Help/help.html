Help Page



Home Page Navigation

The Home page contains features such as a search bar to give you freedom to search what you would like, buttons to have quick access to the menus and buttons located in the top right of the home page

To use the search bar, use your mouse to click within the search bar form and use your keyboard to type a menu item

To use the buttons, hover your mouse over the buttons and left click on your mouse to bring up the relevant menu items, to the right of the menu items you will find a button enabling you to add these items to a basket

In the top right of this page, you will find three buttons, one directing you to this page, one directing you to a login page and one directing you to the Review page

Login Page

The login page allows you to login if you have an existing account, if not you will be directed to a create account page which displays a form allowing you to add your credentials

Review Navigation

This page allows you to write and submit reviews which can then be viewed by accessing the comment section. The comment section displays when the comment was created and who it was created by.

Basket Navigation

The basket page displays what items have been selected allowing you to review what you have added to the basket, from the basket page you can then continue to the checkout page where you can complete your order
